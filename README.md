#  HR-Sales Analytics (Power BI)

##  Overview
This project is a **Power BI Dashboard** built using combined **HR Data** and **Sales Data**.  
It provides insights into **employee attrition, salary distribution, and departmental trends** along with **regional sales performance, product-wise revenue, and profitability analysis**.

##  Dataset
- **HR Data**: Employee demographics, department, salary, attrition  
- **Sales Data**: Region, product category (Headphone, Mobile, Laptop, Tab), sales amount  

## Dashboard Features
- **HR Analytics**
  - Attrition rate by department, age, and gender  
  - Salary vs Attrition correlation  
  - Employee retention insights  

- **Sales Analytics**
  - Revenue by region (Dubai, London, Lahore, Karachi, Islamabad)  
  - Product-wise sales comparison  
  - Monthly/quarterly sales trends  
  - Top-performing regions and products
